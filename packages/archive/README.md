# Archive

This namespace contains packages that are not yet implemented or have been
absorbed into other programs.

Packages are moved here instead of deleted — once a real repository exists,
they will be restored to their appropriate namespace.

## Why a package ends up here

- The program is desktop-internal and not a standalone Store package
- The program was merged into another package
- The program does not have a real repository yet

## Packages in this archive

| Package | Reason |
|---------|--------|
| `apps/builder` | Absorbed into Container Manager — no standalone package |
| `apps/showcase` | Desktop-internal development view — not a Store package |
| `apps/profile` | Desktop-internal (`fs-desktop/crates/fs-profile`) |
| `apps/settings` | Desktop-internal (`fs-desktop/crates/fs-settings`) |
